# pandas-foundation
A curated collection of Jupyter Notebooks documenting my exploration of Pandas and data manipulation techniques. This repository serves as a reference for data preprocessing workflows, which are foundational to building robust AI data pipelines.

## Repository Contents
This collection covers essential data handling techniques, organized to reflect the workflow shown in image_3c50fb.png:

*   **Data Exploration & Manipulation**: Exercises on DataFrame creation, indexing, filtering, and conditional selection.
*   **Merging & Reshaping**: Techniques for joining datasets (inner/left joins) and transforming data using pivot tables and melting.
*   **Data Cleaning & Aggregation**: Handling missing values, removing duplicates, and utilizing `groupby` for custom aggregations.
*   **Feature Engineering**: Application of lambda functions and row-wise calculations to build robust data pipelines.
